# MathOp

## Description

This web application allows users to test their math skills through an interactive quiz. Users can sign up, log in, and take a quiz with dynamically generated questions.

## Features

- User authentication and login system
- Dynamic generation of quiz questions
- Real-time recording of quiz attempts
- Personalized user sessions

## Technologies Used

- HTML, CSS, JavaScript
- Node.js, Express.js
- SQLite
- Bcrypt for password hashing
